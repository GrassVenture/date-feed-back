# DateFeedBack タスク一覧

## 📋 概要

マッチングアプリ等で出会った異性とのデート会話を録音・分析し、フィードバックを提供する Flutter Web アプリケーションの実装タスクを整理します。MVC パターンと HooksRiverpod を採用し、段階的に機能を実装していきます。
